## Introduction

webScraperTables is a collection of Python scripts and notebooks aimed at scraping tables from web pages. It includes scripts for scraping specific websites like Jumia, as well as generic scripts for scraping tables from any website.

## Files

- **datasets/output**: Directory containing output files generated by the scripts.
- **jumia-flash-sales.html**: HTML file generated by the Jumia scraper script.
- **jumiaScr.py**: Python script for scraping tables from Jumia website.
- **jumiaScraper.ipynb**: Jupyter Notebook demonstrating the process of scraping tables from Jumia website.
- **web-scraper-tables.ipynb**: Generic Jupyter Notebook demonstrating the process of scraping tables from any website.
- **wholeScr.py**: Python script for generic table scraping from any website.

## Usage

1. Clone the repository:

```
git clone https://github.com/simlexx-k/webScraperTables.git
```

2. Navigate to the cloned directory:

```
cd webScraperTables
```

3. Run the scripts or notebooks using Python or Jupyter as needed.
